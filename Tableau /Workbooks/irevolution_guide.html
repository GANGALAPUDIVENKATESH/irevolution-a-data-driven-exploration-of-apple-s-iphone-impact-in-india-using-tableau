⚙️ Setup Instructions

1
Connect Data Sources
Open Tableau Desktop. Go to Connect → Text File and load each CSV from the datasets/ folder:
  • iphone_india_sales.csv — Primary sales data
  • india_smartphone_market_share.csv — Competitive market data
  • regional_adoption_india.csv — Geographic data
  • consumer_demographics.csv — Consumer insights
  • apple_india_economic_impact.csv — Economic metrics
2
Create Data Relationships
In the Data Source tab, create relationships between the tables using the Year field as the common key. Use Inner Join for cross-table analysis or keep separate data sources for independent dashboards.
3
Set Geographic Roles
For regional_adoption_india.csv: Right-click the State field → Geographic Role → State/Province. Right-click Region field → Geographic Role → Custom. This enables the India map visualizations.
4
Configure Date Hierarchy
The Year and Quarter fields in sales data should be combined. Create a calculated field: 
MAKEDATE([Year], 
  CASE [Quarter]
    WHEN 'Q1' THEN 1
    WHEN 'Q2' THEN 4
    WHEN 'Q3' THEN 7
    WHEN 'Q4' THEN 10
  END, 1)
Name it Quarter Date and set data type to Date.

🧮 Key Calculated Fields

YoY Growth Rate

// Units Sold YoY Growth
(SUM([Units_Sold_Millions]) - LOOKUP(SUM([Units_Sold_Millions]), -1)) 
  / ABS(LOOKUP(SUM([Units_Sold_Millions]), -1))

Format as Percentage. Use in KPI cards to show growth vs prior year.

Market Share Rank

// Apple's rank among brands for a given year
RANK(SUM([Market_Share_Percent]), 'desc')

Revenue Per Unit (ASP)

// Average Selling Price validation
SUM([Revenue_USD_Billion]) * 1000 / SUM([Units_Sold_Millions])

Penetration Index

// Normalized penetration vs national average
[Penetration_Rate_Percent] / 
  WINDOW_AVG(AVG([Penetration_Rate_Percent]))

Cumulative Sales

// Running total of units sold
RUNNING_SUM(SUM([Units_Sold_Millions]))

🎨 Dashboard Design Specifications

🎨
Color Theme — Apple Dark
Use these hex codes for consistent theming across all dashboards:
  • Background: #1d1d1f
  • Primary Blue: #0071e3
  • India Saffron: #FF9933
  • India Green: #138808
  • Text Primary: #f5f5f7
  • Text Secondary: #86868b
  • Grid Lines: #2d2d2f
📐
Dashboard Sizing
Set all dashboards to Fixed Size: 1440 × 900 px for consistent presentation. For Tableau Public publishing, use Automatic sizing. Font: Use Tableau Book or Calibri (closest available to DM Sans in Tableau).

📤 Publishing to Tableau Public

↑
Server → Tableau Public → Save to Tableau Public
Ensure all data is embedded in the workbook (not live connection). Go to Data → [Data Source Name] → Extract Data before publishing. The embed code from Tableau Public can be pasted into the tableau-embed-frame div in index.html for live embedding.
